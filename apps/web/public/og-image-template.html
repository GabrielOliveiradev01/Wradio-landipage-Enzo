📸 Como Gerar a Imagem Open Graph (1200x630px)

  1. Abra este arquivo HTML no navegador
  2. Personalize o texto acima conforme necessário
  3. Use uma das opções abaixo para capturar:

Opção 1: Screenshot Manual (Mais Simples)

  1. Dê zoom no navegador para 100%
  2. Tire um screenshot da área colorida acima
  3. Recorte para exatamente 1200x630px usando:
    • Preview (Mac): Ferramentas > Ajustar Tamanho
    • Paint (Windows): Redimensionar
    • Ferramentas online: https://www.iloveimg.com/resize-image
  4. Salve como og-image.png na pasta /public

Opção 2: Screenshot com DevTools (Mais Preciso)

  1. Abra Chrome DevTools (F12)
  2. Pressione Cmd+Shift+P (Mac) ou Ctrl+Shift+P (Windows)
  3. Digite "Capture node screenshot"
  4. Clique no container laranja acima
  5. Renomeie para og-image.png e mova para /public

Opção 3: Usar Ferramenta de Design

  1. Canva: Use template "Open Graph" (1200x630)
  2. Figma: Crie frame 1200x630px
  3. Photoshop: Novo documento 1200x630px

Dicas de Design:

  • ✅ Mantenha texto legível (mínimo 40px)
  • ✅ Use cores da marca (laranja: #f97316)
  • ✅ Deixe margem de 50px em todos os lados
  • ✅ Teste em diferentes fundos (claro/escuro)
  • ✅ Inclua logo/nome do produto

Validar Imagem:

  • Tamanho: exatamente 1200x630 pixels
  • Formato: PNG ou JPG
  • Peso: < 1MB (idealmente < 300KB)
  • Testar em: https://www.opengraph.xyz/
